La propriété float sur un élément inline

Qu'est-ce qu'un text wrap?

La propriété float indique qu'un élément doit être retiré du flux normal et doit être placé sur le côté droit ou sur le côté gauche de son conteneur. Le texte et les autres éléments en ligne (inline) entoureront alors l'élément flottant.

Sachant qu'une image est un élément de type inline, des effets intéressants peuvent être créés tel qu'un effet text wrap (le texte semble entourer l'image). L'effet change selon la taille de l'image: si les éléments flottants sont verticalement plus courts que le texte sur lequel ils flottent. Voir MDN.
